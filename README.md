Сервис по продаже вещей кофейной
тематики

Общее описание работы
В данном проекте был реализован сервис по продаже вещей кофейной
тематики. Сервис представлен в виде веб-приложения на Streamlit.
Была реализована авторизация пользователей с разделением ролей на
администраторов сайта и покупателей. Администратор может
регистрировать новых администраторов, добавлять товары, заказывать
товары, менять цену товаров, добавлять поставщиков. Покупатели имеют
возможность покупать товары. Кроме того, у всех пользователей есть
возможность фильтровать товары и узнавать информацию о товарах.
